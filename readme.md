## Some useful commands

-Getting details for command:
`Get-Help -Name 'Write-Output' -Full`

-To create a file:
`New-Item [filename]`

-To open file in VSCode editor:
`code [filename]`

-Promt:
`Write-Output "string. $var etc."`

-Input:
`Read-Host`

-Input with prompting message:
`Read-Host -Prompt "Message"`
